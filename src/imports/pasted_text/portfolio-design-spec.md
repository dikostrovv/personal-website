Animations & Interactions

Use subtle, modern micro-interactions:

1. Expandable Cards (main interaction)
	•	Smooth height expansion (200–300ms)
	•	Content fades in slightly delayed

2. Hover Effects
	•	Cards slightly lift (small translateY)
	•	Shadow increases
	•	Border or background subtly changes

3. Focus Effect
	•	When hovering a card:
	•	other elements slightly fade
	•	active element stays prominent

4. Scroll Reveal
	•	Sections fade in + slight upward motion
	•	Staggered appearance for lists

5. Smooth Navigation
	•	Smooth scroll between sections
	•	Active section highlighted in header
